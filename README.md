## Documentation

### Push
To push a digit to the stacjk you just have to write the digit as is.
```opascal
42 69
```

### Dump
Dump will print the value at the top of the stack followed by a newline. 
```opascal
42 69 dump dump
```
>>>
``` shell
69
42
```
